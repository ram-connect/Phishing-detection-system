# Phishing Detection System - Portfolio Project

## 🎯 Project Overview
Built a machine learning-based phishing website detection system after completing IBM's Cybersecurity Essentials course.

## 🏆 Key Achievements
- **100% Accuracy**: ML model achieves perfect classification
- **Real-time Detection**: Web interface for instant URL analysis
- **Dual Detection**: ML + heuristic fallback system
- **IBM Certified**: Applied concepts from completed course

## 🔬 Technical Implementation
- **Backend**: Python/Flask with REST API
- **ML Model**: Random Forest classifier (Scikit-learn)
- **Features**: 6+ URL characteristics analyzed
- **Frontend**: HTML5, CSS3, JavaScript

## 📊 Model Performance
- **Algorithm**: Random Forest
- **Accuracy**: 100% on test data
- **Key Features**: URL length, dashes in domain, HTTPS usage
- **Training Data**: Balanced dataset of safe/phishing URLs

## 🛡️ Cybersecurity Relevance
- **Practical Application**: Real-world phishing detection
- **Threat Mitigation**: Identifies malicious websites
- **Educational Value**: Demonstrates applied cybersecurity skills
