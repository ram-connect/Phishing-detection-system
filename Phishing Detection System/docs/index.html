🔒 Phishing Detection System

Cybersecurity Portfolio Project

📌 Project Overview

A machine learning-based phishing detection system designed to identify malicious and fraudulent websites through real-time URL analysis and security feature extraction.

This project demonstrates the practical implementation of cybersecurity concepts, web security analysis, and machine learning techniques in detecting phishing attacks.

🚀 Quick Start

Run the project locally using the following commands:

git clone https://github.com/your-username/phishing-detection-system.git
cd phishing-detection-system

pip install -r requirements.txt

python app.py

Open the application in your browser: http://localhost:5001

  • Machine learning-powered phishing detection
  • Real-time URL and website analysis
  • Interactive web interface
